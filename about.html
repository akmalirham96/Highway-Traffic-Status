Highway Traffic Status

Posted on 12 September by Akmal Irham

HTS mobile applications is a application that have been developed to ease the highway users. This applications could help users to avoid being stuck in a heavy traffic. By entering their destination the application will show the best route for them to select, so they could possibly avoid from heavy traffic routes.

By using data that have been post in social media by highway user this website intend to give user the latest highway traffic status.

  • Real time highway traffic status
